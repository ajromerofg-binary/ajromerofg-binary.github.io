HADES — Script de Vigilancia Digital

Sistema automático de vigilancia digital diseñado para proteger a empresas y particulares contra la filtración de datos y ciberdelincuencia. Busca menciones de dominios y emails en Pastebin, verifica brechas en HaveIBeenPwned, escanea repositorios GitHub en busca de credenciales expuestas y consulta inteligencia de amenazas con VirusTotal y Shodan.

Integración con HaveIBeenPwned, VirusTotal, Shodan, Pastebin

HERRAMIENTA DE AUDITORÍA · OWASP TOP 10

WebScan Pro v2.0 — Analizador de Seguridad Web

Escáner de vulnerabilidades web implementado como script Bash autónomo. Cubre los vectores de ataque más frecuentes del OWASP Top 10: SQLi, XSS, XXE, LFI, RFI, Path Traversal, SSRF y SSTI. Genera automáticamente tres informes al finalizar: ejecutivo (dirección), técnico IT y HTML interactivo. Sistema de puntuación de riesgo global 0–100.

12 módulos de detección independientes
Pruebas GET, POST form-urlencoded, POST JSON y cabeceras HTTP
3 informes automáticos por escaneo (ejecutivo, IT, HTML)
Cobertura de 13 motores de plantillas (SSTI) y detección de CMS
github.com/ajromerofg-binary/Web-Scan-Pro →

HERRAMIENTA · AUDITORÍA WINDOWS / ACTIVE DIRECTORY

AD_Audit — Auditoría de Active Directory
